pseudo.ts analytics
// record a set and update PR cache
POST /api/sets { exerciseId, reps, weight }
  → validate() → db.set.create()
  → if weight > currentPR: db.pr.upsert({ value: weight })
  → event("set_logged", { exerciseId }) // emits to chart stream

// time-bucketed query (week)
SELECT date_trunc('week', created_at) AS wk,
       max(weight) AS max_weight
FROM sets WHERE user_id = :uid AND exercise_id = :eid
GROUP BY 1 ORDER BY 1;

RF Asset Finder — Low-Power Gear Tracking

Arduino (C/C++)433 MHz ASK/OOKTimers/Interrupts RadioHeadPython logs

What it does: Tags broadcast ID/seq/status; a base station raises an alarm if a tool isn’t seen within a timeout window.

How I built it: Packet protocol with CRC + backoff/retransmit; sleep modes to save power; range/antenna tests validated reliability.

pseudo.c low-power
// tag main loop (pseudo)
while(true){
  packet = { id, seq++, status, crc(packet) };
  radio_send(packet);
  sleep_ms(250);           // low duty cycle
  if(voltage < THRESH) status = LOW_BATT;
}

// base station
on_receive(pkt){
  seen[pkt.id] = now();
}

if(now() - seen[id] > TIMEOUT){
  alarm("MISSING:", id);
}
